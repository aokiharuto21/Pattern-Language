# デザインシステム：Pattern Language Home

> **採用スタイル：Neo Brutalism（Mono）**  
> 力強く、明快で、遊び心のあるデザイン。

---

## 1. デザイン原則

### 核となる考え方

| 原則 | 説明 |
|------|------|
| **Bold（大胆）** | 太い線、はっきりしたコントラスト、迷いのない表現 |
| **Playful（遊び心）** | アニメーションで「触りたくなる」体験を作る |
| **Clear（明快）** | 余計な装飾を排除し、情報を直接伝える |
| **Unified（統一）** | どの画面でも同じルールに従う |

### ❌ やらないこと

- グラデーション（背景・ボタン）
- ドロップシャドウ（blur系）
- 角丸の多用（ボタン・カードは角丸なし）
- 薄いグレーの文字（コントラストを下げない）

---

## 2. カラーパレット

### 基本：Mono（モノクロ）

```
背景:     #FFFFFF（純白）
テキスト: #000000（純黒）
アクセント: #000000（黒を強調として使う）
```

### 拡張カラー（将来のLanguage View用）

| 名前 | 用途 | カラーコード |
|------|------|-------------|
| **Mono** | デフォルト、ホーム | `#000000` |
| **Fire** | 楽天主義など | `#FF6B35` |
| **Ice** | 将来の拡張用 | `#0066FF` |

> ⚠️ **ルール**: アクセントカラーは**シャドウ**と**ラベル背景**のみに使用。  
> テキストや背景全体には使わない。

## 3. タイポグラフィ

### フォントファミリー

```css
font-family: system-ui, -apple-system, BlinkMacSystemFont, 
             'Segoe UI', Roboto, sans-serif;
```

### サイズと用途

| 用途 | サイズ | ウェイト | スタイル |
|------|--------|----------|----------|
| 見出し（カードタイトル） | 1.5rem | 900 (Black) | UPPERCASE |
| 本文 | 0.875rem | 400 | 通常 |
| ラベル | 0.625rem | 800 | UPPERCASE, tracking: 0.1em |
| ナビリンク | 0.75rem | 800 | UPPERCASE |

---

## 4. ボーダーとシャドウ

### ボーダー

```css
border: 3px solid #000;  /* カード */
border: 2px solid #000;  /* 内部要素、ボタン */
```

### ハードシャドウ（Offset Shadow）

```
通常時:  x: 8px, y: 8px
ホバー時: x: 12px, y: 12px
クリック時: x: 4px, y: 4px
```

> 🎯 **重要**: シャドウはアニメーションさせる。  
> ホバーで広がり、クリックで縮む「押し込む」感覚を演出。

---

## 5. スペーシング

### 基本単位

```
0.25rem (4px)  — 微調整
0.5rem  (8px)  — 小さい間隔
1rem    (16px) — 標準
2rem    (32px) — カード内パディング
4rem+   (64px) — セクション間
```

### カード内部

```
padding: 2rem;
内部要素間: 1rem;
ラベル下マージン: 1rem;
```

## 6. アニメーション

### 使用ライブラリ

```
framer-motion
```

### スプリング設定（基本）

```javascript
const brutalSpring = {
  type: "spring",
  stiffness: 400,
  damping: 15,
};
```

### アニメーションパターン

| 対象 | 動き | タイミング |
|------|------|-----------|
| **カード登場** | 下から上へ + 微回転 | 0.35s, spring |
| **シャドウ** | ホバーで拡大、タップで縮小 | spring |
| **アイコン** | 回転しながら拡大 | 0.3s, delay: 0.15s |
| **テキスト** | フェードイン + 左から右 | 0.2s, delay: 順次 |
| **背景図形** | ゆっくり回転（装飾） | 15-20s, 無限ループ |
| **ヒント矢印** | 左右に揺れる | 1.5s, 無限ループ |

### ホバーとタップ

```javascript
whileHover={{ scale: 1.02, y: -2 }}
whileTap={{ scale: 0.98 }}
```

## 7. コンポーネントの基本パターン

### カード

```
┌─────────────────────────────┐
│ [LABEL]           [#001] ← 番号バッジ
│                             │
│ ┌─────────────────────────┐ │
│ │       ICON AREA         │ │  ← アクセントカラー背景
│ └─────────────────────────┘ │
│                             │
│ TITLE IN UPPERCASE          │  ← font-weight: 900
│                             │
│ Description text here.      │  ← font-weight: 400
│                             │
│ CLICK FOR NEXT →            │  ← ヒント（アニメーション）
└─────────────────────────────┘
        └───────┘
        ハードシャドウ（オフセット）
```

### ボタン

```
┌───────────────┐
│   BUTTON      │  ← 背景白、2px黒ボーダー
└───────────────┘
```

- ホバー: scale 1.05, y -2px
- アクティブ時: 背景を黒に反転

### ナビゲーションリンク

```
LABEL
─────  ← 3pxの下線（アクセントカラー）
```

## 8. 拡張ルール

### 新しい画面を作るとき

1. **同じカラールールを守る**（モノクロ基調 + 1アクセント）
2. **ハードシャドウを使う**（blur禁止）
3. **登場アニメーションを付ける**（spring推奨）
4. **ホバー/タップ反応を付ける**（インタラクティブ要素すべて）

### 新しいLanguageを追加するとき

- 固有のアクセントカラーを1色定義
- シャドウとラベルにのみ適用
- 他のルール（ボーダー、タイポグラフィ）は変えない

---

## 9. 禁止事項チェックリスト

- [ ] ドロップシャドウ（box-shadow with blur）を使っていないか
- [ ] グラデーション背景を使っていないか
- [ ] 薄いグレー（opacity < 0.5）の文字を使っていないか
- [ ] 角丸（border-radius > 0）をカードやボタンに使っていないか
- [ ] アニメーションなしでインタラクティブ要素を作っていないか
