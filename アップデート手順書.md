### 🔄 今後のアップデート方法

コードを変更したら、以下のコマンドを実行するだけで自動デプロイされます：

```bash
git add .
git commit -m "変更内容のメッセージ"
git push
```

### 📦 新しいLanguageを追加する場合

1. `assets/{language_id}_patterns.json` を作成
2. `assets/illustrations/{language_id}/` にイラスト配置
3. 上記のgitコマンドでプッシュ → 自動デプロイ！
